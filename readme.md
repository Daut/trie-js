# Trie.ts

Trie data structure implementation in TypeScript.

## Usage

### Create new Trie
```typescript
const Trie = require('trie');
...
const trie = new Trie();
```

### Insert element
`trie.insert('Tree');`

### Search for an element
`trie.search('Tree');`

## Build

Run `npm run build` to build project.
